# 🌐 Bootstrap Responsive Website

A multi-section responsive website built with Bootstrap 5, demonstrating grid layouts and component-based design.

## 🔗 Live Demo
[View Live](https://haneen562.github.io/bootstrap-Project/)

## ✨ Features
- Fully responsive layout using Bootstrap 5 Grid
- Navigation bar with mobile hamburger menu
- Multiple sections: Hero, About, Services, Contact
- Reusable Bootstrap components (cards, buttons, forms)

## 🛠️ Tech Stack
- HTML5
- CSS3
- Bootstrap 5


## 🚀 How to Run
1. Clone the repo: `git clone https://github.com/haneen562/bootstrap-Project`
2. Open `index.html` in your browser
